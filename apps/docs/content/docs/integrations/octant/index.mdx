# Octant V2 Integration

[Octant V2](https://docs.v2.octant.build/) is yield-based public goods funding infrastructure from Golem Foundation. It shares core design principles with Curator Studio: ERC-4626 yield sources, principal preservation, and programmable distribution.

## Architecture Overlap

| Curator Studio             | Octant V2                     |
| -------------------------- | ----------------------------- |
| YieldRedirector4626        | Funding Vaults                |
| Strategy (weighted splits) | Routing & Splitting Contracts |
| SplitsWarehouse            | Distribution infrastructure   |

**Key difference**: Octant includes allocation mechanisms (QF, QV) and community staking. Curator Studio focuses on curator-defined allocations and fund-of-funds.

## Collaboration Opportunities

Rather than competing infrastructure, these protocols can compose:

1. [Strategy as Recipient](/docs/integrations/octant/strategy-as-recipient) - Curator Strategy receives Octant yield
2. [Allocation Mechanisms](/docs/integrations/octant/allocation-mechanisms) - Use Octant QF/QV in Curator flows
3. [Shared Vaults](/docs/integrations/octant/shared-vaults) - Octant vaults, Curator allocation
4. [Fund-of-Funds](/docs/integrations/octant/fund-of-funds) - Cross-protocol hierarchies
5. [Recipient Registry](/docs/integrations/octant/recipient-registry) - Shared project verification
